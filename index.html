Apex Social Media

Social Media Application where you can POST pictures, LIKE your friends' pictures and DELETE your pictures if required
Project

3 KEY Features:

Adding a Post

This feature allows the user to add a post to their timeline.

Liking a Post

This feature allows the user to like a post.

Deleting a Post

This feature allows the user to delete a post.

ABOUT
